# Redux

- Store --> Kho chứa các state global
- Reducer --> Hàm cập nhật State
- Action --> Object mô tả hành động được gửi từ Component lên Reducer
- Dispatch --> Hàm gửi action lên Reducer
- Subscribe --> Hàm lắng nghe sự thay của state
