# SCSS

## Trình biên dịch

npm i sass -g

## Biên dịch cả folder

sass tenfolderscss:tenfoldercss

Watch Mode

sass tenfolderscss:tenfoldercss --watch

## Biên dịch 1 file

sass duong-dan-file-scss duong-dan-file-css

## Nén file CSS (.min)

sass duong-dan-file-scss duong-dan-file-css --style compressed
